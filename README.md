# 短信转发助手 (SmsForwarder)

这是一个专门为安卓手机（特别适配了小米 HyperOS/MIUI）编写的轻量级短信转发应用。

## 🌟 核心功能
*   **双卡支持**：自动截获手机接收到的所有短信，不限双卡。
*   **关键字过滤**：支持自定义关键字（默认留空，即转发所有短信）。
*   **双通道推送 (支持自定义地址)**：
    *   **NTFY**：可自定义推送地址 (默认配置为 `https://minli52129.onrender.com/duanxin`)。
    *   可以自定义，比如 https://minli52129.onrender.com/88888888 。这个8888888就是你自己的通道。
    *   然后用网页访问这自定义地址查看提醒。或者安装ＮＴＦＹ应用，添加你的主题地址就行。无限量通知。
    *   **飞书机器人**：可自定义飞书 webhook 地址 (默认留空，不转发到飞书)。
*   **本地日志**：内置本地 SQLite 数据库，可直接在 App 首页查看最近的短信转发成功/失败记录。

## 📱 小米 / HyperOS / MIUI 防杀配置指南 (必看)。只测试了小米，其他没试。安卓可以用。苹果华为不行。
由于小米系统对后台应用的限制非常严格，为了确保在息屏和后台状态下依然能够正常监听和转发短信，安装 App 后**务必按照以下步骤配置**：

1. **授予基础短信权限**
   *   打开 App 主界面，点击“检查/请求常规短信权限”，并在弹出的系统提示中允许。
2. **授予通知类短信权限 (极其重要)**
   *   点击“打开权限设置 (请手动允许通知类短信)”，跳转到应用详情页。
   *   点击“权限管理”，找到“通知类短信”或“服务类短信”，将其设置为**允许**（如果收不到验证码或银行通知，多半是因为这个权限没开）。
3. **忽略电池优化**
   *   在 App 首页点击“设置忽略电池优化 (防杀)”，在系统弹窗中选择“无限制”。
4. **开启自启动**
   *   点击“打开小米自启动设置”，将其加入自启动白名单。
5. **多任务界面加锁 (终极防杀)**
   *   从手机底部上滑进入“最近任务（多任务）”列表。
   *   长按“短信转发”App 的卡片。
   *   点击弹出的**锁头图标**，将其锁定（卡片旁边会出现一个小锁）。这样在点击“一键清理”时，该应用就不会被杀掉了。

## 🛠 开发与编译
本项目使用 Kotlin 和 Android SDK 34 开发。

1. 克隆代码：
   ```bash
   git clone https://github.com/minli52129/SmsForwarder.git
   ```
2. 本地编译：
   ```bash
   ./gradlew assembleDebug
   ```
编译生成的 APK 位于 `app/build/outputs/apk/debug/app-debug.apk`。
